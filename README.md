# 대학교 수강신청하기 웹사이트 만들기
- 5인 프로젝트
- 제작기간 약 15일
- 학생 - 교수 대학교 수강신청 / 출석체크 웹페이지 만들기
  1. 학생 - 수강신청 / 내가 신청한 강의 보기 / 즐겨찾기 추가하기 / 만족도 설문조사
  2. 교수 - 학생 출석체크 및 출석확인 / 내강의 보기 / 학생 수강신청 화면 접속불가
  3. 공통 - 로그인 / 회원정보 변경 / 회원가입
- 이용 프로그램
  1. DB : MySQL
  2. Front : VSCODE
  3. Back : JAVA 11 (STS 3버전, JSP - Servlet 구현)
  4. 설계도 : Figma
